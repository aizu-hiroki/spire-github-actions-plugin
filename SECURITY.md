# Security Policy

## Important Notice

This project is **experimental** and has **not been audited for security**.
It is not recommended for production use without thorough review.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

1. **Do NOT open a public issue.**
2. Send a report via [GitHub Security Advisories](https://github.com/aizu-hiroki/spire-github-actions-plugin/security/advisories/new).

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
